# Communication Facilitator Templates

Use these artifacts only when they reduce ambiguity or coordination risk. Do not maintain every artifact by default.

## Full Status Update

```text

Current Objective:
Current Action:
Status:
Output Produced:
Needs:
Needed From:
Reason Needed:
Deadline / Urgency:
Blockers:
Risks:
Next Step:
Handoff Target:
Definition of Done:
```

## Handoff Record

```text
From:
To:
Context:
Output Delivered:
Open Questions:
Known Risks:
Required Next Action:
Completion Criteria:
```

Use for cross-team, cross-tool, or cross-agent handoffs where context loss is likely.

## Escalation Message

```text

```

## Clarifying Questions

Use these prompts to tighten ambiguous communication:

- Who specifically needs to act?
- What exactly is needed?
- What artifact or decision is missing?
- Why does the next agent need it?
- When is it needed?
- What is the next observable action?
- What output will unblock the next agent?
